General Workflow
*Check your status at anytime
$ git status
$ git pull origin master [Get the most updated changes]
$ git branch <branch_name>
$ git checkout <branch_name>
Program...
Whenever new files are added
$ git add .
When you make changes
$ git commit -m "Message" [Small commits, often. Easier for you, especially for co-programmers.]
When you're ready to push commits to the local repository
$ git push origin <branch_name>
Once your branch changes are approved, you could merge onto master.
$ git merge <branch_name>
$ git branch -d <branch_name>
